FAQ

Common questions before ordering

Basic information customers usually need before requesting a website quotation.

What do I need to prepare?

Please prepare your name/brand, short introduction, photo or logo, contact details, social links, service details, project details or any content you want to include.

Is domain included?

Basic publishing using GitHub Pages is included. Custom domain setup can be added, but the domain purchase fee is not included in the package price.

Can I request changes after the website is done?

Yes. Each package includes minor revisions. Extra changes, new sections or major redesign requests may require additional charges.

Is this suitable for online portfolio or internship application?

Yes. The CV/Portfolio package is suitable for students, interns, fresh graduates and job seekers who want a clean website link to share with employers.
