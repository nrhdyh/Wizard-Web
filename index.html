FAQ

Common questions before ordering

Basic information students usually ask before creating a GitHub.io portfolio.

Do I need a GitHub account?

Yes. You need your own GitHub account so the portfolio link can use your own username, such as yourusername.github.io.

Do I need to share my GitHub password?

No. You do not need to share your password. You only add me as a collaborator to help upload and set up the portfolio files.

Is there any monthly hosting fee?

No. The portfolio will be published using GitHub Pages, so there is no monthly hosting fee for this package.

Can I use the link in my resume or internship application?

Yes. You can add the GitHub.io link to your resume, LinkedIn profile, email signature, internship form or job application.

What do I need to prepare?

Please prepare your resume PDF, short profile, education details, skills list, project details, project screenshots if available, contact links and preferred design style.

Can I update my portfolio later?

Yes. Future updates can be requested as an add-on, such as adding new projects, updating resume files, changing profile details or adding new links.

Is this suitable for systems with login or database?

No. This package is for static online CV and portfolio websites only. It does not include login systems, databases, payment gateways or advanced web application functions.
